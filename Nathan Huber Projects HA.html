Home Automation

During my free time I like to design and build various things and my largest project this far is my Home Automation project. It is using an Arduino Nano as the micro-controller and communicating over a 2.4gHz network to all of the other modules. These modules send the inside, outside and aquarium temperatures back to the main module when requested. As well as control my garage door, front door, aquarium lights, and my bedside light. The data is then displayed on an LCD screen and can be controlled on a secure App I have made. I am currently working on using a smaller micro-controller and networking the system over WiFi. Soon I hope to start making circuit boards for them.

The main module shown above displays the military time and date, along with the requested temperatures and when my alarm is set. If the other modules are not found (not turned on or signal interference) the module will not display any data and move the Alarm Status up a line. It will continue to look for the other modules even though it is not displaying data for the them. For example, that way if the fish tank module was able to connect it will automatically refresh on the main unit. The system can also be controlled via an APP I made. The APP communicates through Bluetooth to the main unit to send and receive data.

Shown below is the insides of the Main Unit. The Photocell off to the side monitors the ambient light intensity and adjusts the LCD brightness accordingly. The module shown below is one of the first ones I ever made to communicate back to the main unit.

The module shown above monitors the inside temperature and reports back to the main unit when requested. Also, the Red & Black Terminal Posts are connected to a set of N.O. contacts on a relay. The contacts are connected in parallel to the button that opens the main entrance of my apartment building. This allows me to buzz people in from my phone. I also control the lighting in my aquarium and monitor the temperature of the tank.

The lighting is a strip of RGB LED's powered from the Module. I can adjust the lighting through my APP and it will turn on at specific times or when the Aquarium Module monitors the light in the room and if it is dark, it will turn on. The LED's are driven by 3 MOSFETs controlled by PWM Outputs on the Arduino Nano. Another module still under debug and programming is the Garage Door Module.

The Garage Door Module allows me to control my two-bay garage. I am in the process of having it detect when the door is open or closed using either prox's or ultra sonic range sensors. To open the doors, the app is password protected to be able to control any doors.

The user can create an account, change there password if they so choose. Also, underneath the settings, the user can adjust what functions are available and change the colors of the background and buttons.
